### References

- https://github.com/guoruoqian/DetNet_pytorch


### Prerequisites

- Python 2.7 or 3.6
- Pytorch 1.1 or higher
- CUDA 8.0 or higher
- tensorboardX

### Preparation

First of all, clone the code
```
git clone https://github.com/leowangzi/DetNet.git
cd DetNet
```
Then, create a folder:
```
mkdir data
```
or
```
ln -s [source_data] data
```
